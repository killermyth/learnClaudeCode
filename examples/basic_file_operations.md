# 基础文件操作示例

## 1. 创建文件

### 创建简单文本文件
```
使用 create_file 工具
指定文件路径和内容
```

### 创建代码文件
```
创建 TypeScript 文件
包含类型定义
添加导入声明
```

## 2. 读取文件

### 读取特定范围
```
指定 startLine 和 endLine
获取需要的内容
```

### 获取完整文件
```
读取整个文件内容
理解代码结构
```

## 3. 修改文件

### 替换字符串
```
include 3-5 行上下文
确保唯一匹配
```

### 批量修改
```
使用 multi_replace_string_in_file
同时进行多个替换
```

## 实践建议

1. 在修改前先读取文件内容
2. 确保修改的唯一性
3. 保留足够的上下文信息
4. 验证修改结果
