# 代码搜索示例

## 1. 文件搜索

### 按文件名搜索
```
使用 file_search 工具
提供 glob pattern
找到匹配文件
```

### 常见模式
```
**/*.ts       - 所有 TypeScript 文件
src/**        - src 目录下所有文件
**/index.ts   - 所有 index.ts 文件
```

## 2. 文本搜索

### 使用 grep_search
```
指定搜索模式
包含文件过滤
获取匹配结果
```

### 使用 semantic_search
```
自然语言查询
理解代码含义
找相关实现
```

## 3. 代码分析

### 使用 list_code_usages
```
查找符号定义位置
追踪符号使用
理解代码依赖
```

### 使用 get_errors
```
获取错误信息
定位问题位置
验证代码正确性
```

## 实践建议

1. 精确指定搜索范围
2. 利用正则表达式提高精度
3. 使用语义搜索理解代码意图
4. 追踪代码使用情况
